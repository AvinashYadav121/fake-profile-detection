# Instagram Fake Account Detection System

This project implements an end-to-end machine learning pipeline for detecting fake Instagram accounts.

Features:
- Automated Instagram profile data extraction
- Advanced feature engineering (log scaling, entropy, ratio analysis)
- Interactive labeling UI (Real / Fake classification)
- Custom dataset builder
- XGBoost model training pipeline
- Scalable architecture for continuous retraining

Tech Stack:
- FastAPI
- React
- Instaloader
- Pandas
- XGBoost
- Scikit-learn

This system demonstrates real-world ML workflow including data collection, labeling, feature engineering, model training, and deployment readiness.
